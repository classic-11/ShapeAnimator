# ShapeAnimator

\# 🎮 Shape Animator



A WPF desktop application built with C# and .NET 10



\## Features

\- ✅ Draw Circles, Rectangles and Triangles

\- ✅ Shapes bounce off all 4 walls

\- ✅ Play / Stop animation controls

\- ✅ Two-click drawing system



\## How to Run

1\. Clone the repository

2\. Open ShapeAnimator.sln in Visual Studio

3\. Press F5 to run



\## Tech Stack

\- C#

\- WPF

\- .NET 10

\- Visual Studio
